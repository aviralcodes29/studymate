# 📚 StudyMate – Responsive Student Planner

StudyMate is a responsive student productivity web application built using HTML, CSS, and JavaScript. It helps students organize study tasks, manage deadlines, track progress, and improve productivity through a clean and user-friendly interface.

This project was developed as part of **DecodeLabs Full Stack Development – Project 1**, focusing on building a responsive frontend interface using modern web development fundamentals without any frameworks.

---

## 🌐 Live Demo

🔗 https://aviralcodes29.github.io/studymate-responsive-planner/

---

## 📖 Project Overview

StudyMate is designed to help students plan their daily academic activities efficiently. Users can add tasks, set deadlines, assign priorities, track completion status, and monitor their overall study progress.

The project follows a **mobile-first responsive design approach** and demonstrates the practical use of:

- Semantic HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Design
- JavaScript DOM Manipulation
- LocalStorage
- Accessibility Best Practices

---

## ✨ Features

### 📝 Task Management

- Add new study tasks
- Assign subjects to tasks
- Set task deadlines
- Delete tasks
- Mark tasks as completed
- Undo completed tasks

### 🔍 Search & Filter

- Search tasks by name
- Search tasks by subject
- Filter tasks by:
  - All Tasks
  - Pending Tasks
  - Completed Tasks

### 🎯 Priority System

- High Priority
- Medium Priority
- Low Priority

### 📊 Progress Tracking

- Total Tasks Counter
- Completed Tasks Counter
- Pending Tasks Counter
- Dynamic Progress Bar

### 🎨 User Experience

- Fully Responsive Layout
- Mobile Navigation Menu
- Dark Mode / Light Mode Toggle
- Clean Modern UI
- Smooth User Interaction

### 💾 Data Persistence

- LocalStorage Support
- Tasks remain saved after page refresh
