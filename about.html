About StudyMate

StudyMate is a responsive student planner web application built using HTML, CSS, and JavaScript. It helps students manage study tasks, deadlines, priorities, and progress.

This project is created as DecodeLabs Full Stack Development Project 1. The main goal of this project is to practice responsive frontend development using mobile-first design, semantic HTML, CSS Flexbox, CSS Grid, and JavaScript DOM manipulation.

The project is built without using any frontend framework. It focuses on mastering web development fundamentals and creating a clean, accessible, and user-friendly interface.

Project Type

Responsive Frontend Web Application

Built With

HTML5, CSS3, JavaScript

Main Focus

Responsive UI, DOM Manipulation, LocalStorage
